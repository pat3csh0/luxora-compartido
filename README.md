# luxora-compartido

Recursos, snippets y utilidades de uso libre para usuarios de **Luxora Digital** (white-label de GoHighLevel / GHL).

Pensado para que cualquier usuario de Luxora pueda copiar el código de este repo y pegarlo en su landing, funnel o formulario sin necesidad de saber programar.

## Inventario

| Carpeta | Qué contiene |
|---|---|
| [`verificador-email/`](verificador-email/) | Snippet que detecta typos en el dominio del email (`hormail.com`, `gmail.con`, `yahoo.esr`, etc.) y sugiere la corrección al usuario antes de que envíe el formulario. Dos variantes: texto oscuro (light) y texto claro (dark). |

## Cómo usar este repo

1. Entra en la carpeta de la utilidad que te interesa.
2. Lee el `README.md` de esa carpeta (cada una tiene instrucciones de instalación específicas).
3. Copia el código y pégalo donde te indique.

No necesitas clonar el repo ni saber Git. Todo está pensado para copiar/pegar.

## Contribuir

¿Tienes un snippet útil que quieras compartir con otros usuarios de Luxora? Abre un Pull Request con:

- Una carpeta nueva con el nombre de la utilidad.
- El archivo `.html` o `.js` listo para copiar y pegar.
- Un `README.md` explicando qué hace, dónde pegarlo y cómo probarlo.
- Un par de capturas o GIFs si ayudan.
